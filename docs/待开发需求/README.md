# 待开发需求

本目录存放**尚未开发或开发中**的需求文档。

## 文档规范

- 文件名使用**中文**，建议格式：`<模块>-<需求名>.md`（如 `仓储-盘点功能.md`）
- 建议包含以下内容：
  - 需求背景
  - 功能描述
  - 验收标准
  - 优先级（高 / 中 / 低）
  - 关联模块与依赖
  - 备注

## 生命周期

- 新需求创建于此目录
- 开发完成后，将业务说明性质的文档移至 [业务说明](../业务说明/)
- 技术实现方案文档移至 [项目文档](../项目文档/)

## 示例文件名

- `AGV-任务取消需求.md`
- `打印-模板动态配置.md`
- `仓储-盘点功能.md`
